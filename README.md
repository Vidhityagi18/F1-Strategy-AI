# 🏎️ F1 Strategy AI

An AI-powered Formula 1 race strategy platform built using real telemetry data.

## Goals

- Analyze historical Formula 1 race data
- Predict optimal pit-stop timing
- Recommend tire strategies
- Visualize telemetry
- Build an AI race engineer

## Tech Stack

- Python
- FastF1
- Pandas
- NumPy
- Scikit-learn
- FastAPI
- React
